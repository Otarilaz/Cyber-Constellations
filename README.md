# Cyber-Constellations

Bounding Box Constellation Generator

This application simulates the effect of bounding boxes from computer vision but replaces standard labels with various concepts from philosophy of consciousness, cybernetics, and the creative works of Kenji Siratori.
Features:
    Generate a "constellation" of bounding boxes with philosophical and cybernetic terms.
    Overlay bounding boxes on an image of your choice.
    Customize the number of boxes, lines, and text variations.
    Upload and use your own fonts for a unique aesthetic.

This project blends AI aesthetics with philosophical themes, creating an abstract and thought-provoking visual experience.

Генератор Созвездий из Bounding Box

Приложение имитирует эффект bounding box (рамок из компьютерного зрения), но вместо дат или стандартных меток подставляет концепции из философии сознания, кибернетики и творчества Кэндзи Сиратори.
Возможности:
    Генерация «созвездий» из рамок с философскими и кибернетическими понятиями.
    Наложение рамок поверх загруженного изображения.
    Настройка количества боксов, линий и текста.
    Загрузка и использование собственных шрифтов для уникального стиля.
